# Herramientas externas

ShortsFactory actua como orquestador local. Las herramientas externas son opcionales:

- ElevenLabs: voz natural y timing/alignment si la API lo devuelve.
- Higgsfield: prompt packs manuales para clips generativos.
- Picsart: planes manuales de procesado y API preparada como futura opcion.

La pantalla `Herramientas externas` permite revisar estado, generar prompt packs, importar clips, revisar licencias y ver costes.

Reglas:

- Sin API keys la app no debe romperse.
- No se sube nada a YouTube.
- No se descargan clips/musica protegidos.
- Todo asset externo debe tener licencia y uso comercial confirmado antes de render/export.
