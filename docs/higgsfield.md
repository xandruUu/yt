# Higgsfield

La primera integracion es manual/handoff.

ShortsFactory genera:

- `README_HIGGSFIELD.md`
- `master_prompt.md`
- `negative_prompt.txt`
- `visual_style.md`
- `scene_prompts/scene_XX.txt`
- `scene_prompts.json`
- `voiceover_script.txt`
- `shot_list.csv`

Uso:

1. Genera y aprueba un `VisualPlan`.
2. Abre `Herramientas externas > Higgsfield`.
3. Genera el prompt pack.
4. Copia prompts en Higgsfield.
5. Descarga clips.
6. Importa clips en ShortsFactory y revisa licencias.

La integracion MCP queda como placeholder seguro hasta que se active explicitamente.
